# Basic area
The area chart is a good choice to visualize **trends** and [comparisons](https://smartvikisogn.github.io/HChartsCatalog/webpages/comparison.html) using a quantitative/volume dimensions. This demo compares the nuclear stockpiles volume over time between USA and USSR.
 
#### Tip
Notice the use of the [pointStart](http://api.highcharts.com/highcharts/plotOptions.area.pointStart) feature to avoid having to set an X value for each point.
